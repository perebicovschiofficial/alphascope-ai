# AlphaScope AI — Next.js Starter

Premium landing page starter for a global AI trading membership brand.

## Includes
- Hero section
- Markets section
- Pricing section
- Referral section
- Future dashboard preview
- Dark fintech styling

## Run locally
```bash
npm install
npm run dev
```

## Deploy to Vercel
1. Upload this folder to a GitHub repository
2. Go to Vercel
3. Import the GitHub repo
4. Click Deploy
